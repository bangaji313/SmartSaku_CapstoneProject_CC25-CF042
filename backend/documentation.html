ğŸ“„ Dokumentasi API Transaction

1. ğŸ“¥ Get Transactions

Endpoint: GET /api/users/{userId}/incomes atau GET /api/users/{userId}/expenses

Deskripsi: Mengambil daftar transaksi berdasarkan jenis (pemasukan atau pengeluaran).

Response Body:

[
    {
      "nominal": 100000,
      "kategori": "Transportasi",
      "deskripsi": "Ojek coba",
      "_id": "68444fc18bdfc9598b95aeb2",
      "date": "2025-06-07T14:42:09.468Z"
    }
  ]

Keterangan:

  • nominal: Jumlah uang
  • kategori: Kategori transaksi
  • deskripsi: Keterangan transaksi
  • date: Tanggal transaksi

2. â• Add Transaction

Endpoint: POST /api/users/{userId}/incomes atau POST /api/users/{userId}/expenses

Deskripsi: Menambahkan transaksi pemasukan atau pengeluaran untuk user.

Request Body:

{
    "nominal": 100000,
    "kategori": "Transportasi",
    "deskripsi": "Ojek coba"
  }

Response Body:

{
    "message": "Transaction added",
    "transaction": {
      "nominal": 100000,
      "kategori": "Transportasi",
      "deskripsi": "Ojek coba"
    }
  }

Keterangan:

  • transaction: Data transaksi yang berhasil ditambahkan
